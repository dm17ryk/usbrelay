# UsbRelay GUI Asset Pack

A tailored icon and splash-screen set for a GUI application that controls USB relay boards.

## Visual direction

The design is based on the hardware style from the supplied reference:
- red PCB
- blue Songle-style relay blocks
- green screw terminal blocks
- silver USB connector
- USB trident + yellow relay/electric switching mark

## Included

- `icons/usbrelay.ico` — Windows icon bundle
